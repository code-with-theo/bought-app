# 买过｜页面规格

## 全局框架

- 桌面端内容容器最大宽度 720px；移动端 375px 宽度下完整可用。
- 底部固定导航：首页、物品库、居中添加、复购、我的。添加为主要视觉锚点。
- 每页内容底部预留底部导航高度 + 安全区；触控目标不小于 44px。
- 加载使用骨架屏；错误状态说明发生了什么，并给出重试或返回操作。

## 首页 `/`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 让用户一眼看到购物记忆的价值，并快速继续记录或查看待评价商品。 |
| 信息优先级 | 欢迎语/App 名称 → 搜索入口与数量 → 值得复购 → 最近记录 → 等待评价 → 添加按钮。 |
| 主要操作 | 搜索、进入详情、查看复购、去评价、记录商品。 |
| 空状态 | “还没有买过的记录。下次买到喜欢的，记在这里吧。” + 记录商品。 |
| 加载状态 | 数量、横向卡片和列表以对应尺寸骨架显示。 |
| 错误状态 | “这页暂时没打开，试着重新加载。” + 重试。 |
| 移动端细节 | 横向复购卡片可滑动；最近记录为紧凑列表；悬浮添加按钮不遮挡最后一项。 |
| 验收 | 有数据时四个内容区按规则展示；无数据时无空白断层；点击任一商品进入正确详情。 |

## 添加/编辑 `/products/new`、`/products/[id]/edit`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 十几秒完成一次基础记录；编辑时安全更新原记录。 |
| 信息优先级 | 返回/标题 → 名称、照片、分类 → 保存 → 可展开选填。 |
| 主要操作 | 输入名称、选择分类、上传/替换/删除图片、展开选填、保存。 |
| 空状态 | 照片区显示“拍张照，方便下次认出来”。 |
| 加载状态 | 图片处理时预览区显示处理中，保存按钮禁用并显示“正在保存”。 |
| 错误状态 | 按字段显示必填、图片格式、图片处理、存储失败错误；不清空用户已填内容。 |
| 移动端细节 | 保存按钮始终可见或置于表单末端；键盘弹起时仍可提交；日期和数字键盘合适。 |
| 验收 | 仅填三项必填即可保存；编辑保留旧值；图片替换与删除可用；失败后可重试。 |

## 商品详情与评价 `/products/[id]`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 快速回忆这是什么、用得如何、还买不买。 |
| 信息优先级 | 照片/名称/分类 → 复购结论 → 星级/评价 → 购买详情 → 编辑/删除。 |
| 主要操作 | 评分、选择复购结论、输入评价、点快捷标签、编辑、删除。 |
| 空状态 | 记录不存在时显示“这条记录可能已经被删除了。” + 返回物品库。 |
| 加载状态 | 顶部图片和信息区骨架；评价控件不可操作。 |
| 错误状态 | 评价保存失败提示“没保存成功，再试一次”；保留输入内容。 |
| 移动端细节 | 三个结论按钮等宽且可单手点击；星级和标签允许换行；删除与编辑不挤占主要评价区。 |
| 验收 | 复购三选一明显高于普通字段；任意评价修改刷新后仍在；删除必经二次确认。 |

## 物品库 `/library`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 快速确认是否买过，以及值不值得再买。 |
| 信息优先级 | 搜索 → 快捷筛选 → 筛选/排序/视图 → 结果。 |
| 主要操作 | 搜索、快捷筛选、打开筛选面板、排序、切换网格/列表、清除筛选、进入详情。 |
| 空状态 | 无记录：“从第一件开始，给购买留个记号。”；无结果：“没找到这件，换个词试试。” + 清除筛选。 |
| 加载状态 | 搜索框下显示卡片/列表骨架。 |
| 错误状态 | “物品库暂时没加载出来。” + 重试。 |
| 移动端细节 | 筛选使用底部抽屉；选择项足够大；网格至少两列且文字截断不溢出。 |
| 验收 | 搜索、分类、结论、评分、排序可组合；清除后返回默认；网格与列表信息均完整。 |

## 复购 `/repurchase`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 只看明确标记“会复购”的商品。 |
| 信息优先级 | 标题和解释 → 复购商品列表。 |
| 主要操作 | 进入详情、转到物品库。 |
| 空状态 | “还没有认定会回购的。用过几次，再来做决定。” + 查看物品库。 |
| 加载状态 | 列表骨架。 |
| 错误状态 | “复购清单暂时没打开。” + 重试。 |
| 移动端细节 | 复用物品库紧凑列表，无额外复杂筛选。 |
| 验收 | 结果严格等于 repurchase 结论；刷新后同步更新。 |

## 我的 `/me`

| 项目 | 规格 |
| --- | --- |
| 页面目标 | 回顾记录并管理本地数据。 |
| 信息优先级 | 总数/会复购/避雷统计 → 导出 → 清除演示数据 → 清空全部 → 关于。 |
| 主要操作 | 导出、清除演示数据、清空全部数据、查看关于。 |
| 空状态 | 统计显示 0；保留导出入口但禁用并说明“还没有可导出的记录”。 |
| 加载状态 | 统计卡片骨架。 |
| 错误状态 | 导出失败显示原因与重试；清除失败不改变界面统计。 |
| 移动端细节 | 危险操作置于页面底部、视觉降级；确认弹窗按钮不并列误触。 |
| 验收 | 三项统计正确；导出产生可读文件；两类清除范围准确、均二次确认。 |
